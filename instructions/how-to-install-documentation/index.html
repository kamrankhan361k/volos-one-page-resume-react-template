Getting Started

Volos is an elegant personal CV/Resume portfolio React template, It comes with great UI/UX and is easily customized.

We will start by installing the documentation of the template first, and after installing the documentation you will find detailed information and instructions for the template.

You can use the online documentation if you want to skip this part: online documentation


Recuirments

Make sure you have nodejs and npm installed in your device

You can check for them after installation by writing npm -v and node -v in the cmd

install node

npm -v
9.1.3      `npm version we used`

node -v
v18.12.1     `nodejs version we used`


Install npm Packages

Open App folder directory (./instructions/documentation) by a code editor like vscode and write in the terminal `npm install` to install documentation dependencies

npm install


Development server

After installation you can run `npm start` command to see the documentation alive (http://localhost:3000/volos-react-docs/)

npm start
